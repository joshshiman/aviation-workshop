## đź”„ How to Change Theme

### For Official Themes (Easy)

1. Edit `_config.yml`:
   ```yaml
   theme: jekyll-theme-THEMENAME
   ```

2. Commit and push:
   ```bash
   git add _config.yml
   git commit -m "Change theme to THEMENAME"
   git push origin main
   ```

3. Wait 2-3 minutes for rebuild

### Examples:
```yaml
theme: jekyll-theme-minimal
theme: jekyll-theme-slate
theme: jekyll-theme-hacker
theme: jekyll-theme-architect
```
